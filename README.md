# Battleship
Battleship built using TDD testing and webpack.
HTML page set up,
Javascript with classes for ship, player, and gameboard.
Single player game versus computer, interactive ship placement, and turn-based gameplay.
CSS for styling. Project of The Odin Project.

How to Play:

First place your ships.
Click and drag to the Player Board.
You can click "Flip" to make your ships vertical.
Once all ships are placed click "Start".
Select a cell on the Computer Board to attack.
Wait until the computer finishes attacking and select a cell again.
Continue this until the computer or you sinks all the opponents ships.
At any time you can click "Reset" to reset the game.
